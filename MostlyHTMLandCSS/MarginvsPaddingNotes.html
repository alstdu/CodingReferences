What is Margin

Margin is used to create space around elements, outside of any defined borders.

Margin Values

Margin with one number will apply the same value to all four sides of the elements

Margin with two numbers will say the first value is for the top and bottom and the second value is for the left and right side.

Margin with three numbers will apply the first value to the top, the second number to the left and right, and the third value for the bottom

Margin with four numbers will apply each value going clockwise. The first for the top, the second for the right, the third for the bottom, and the fourth for the left.

A positive number indicates it's further away from the element. A negative number will indicate it's closer to the element

What is Padding

An element's padding area is the space between its content and its border. on the inside. it squishes its contents inward.

Padding Values

Padding with one number will appy the same value to all sides of the element

Padding with two numbers will say the first value is for the top and bottom and the second value is for the left and right side.

Padding with three numbers will apply the first value to the top, the second number to the left and right, and the third value for the bottom

Padding with four numbers will apply each value going clockwise. The first for the top, the second for the right, the third for the bottom, and the fourth for the left.

The Difference Between Padding & Margin

Margin creates space around the container or elements. Padding is used to create space inside the container. Padding is like squishing tofu into a container. Margin is like two magnets repelling each other.

Viewing on Dev Tools

right click on the document, inspect, and you can hover over different elements and containers and see whether or not the elements have padding (displayed with green) or a margin (displayed with orange).
