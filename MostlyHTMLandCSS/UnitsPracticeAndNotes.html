The Difference Between CSS Units

Pixels has a fixed width, and percent is what percentage smaller than the width of the parent. for example here our parent is 135px wide. you can see 100px takes up exactly 100px. and the 50% takes up exactly 50% of 135 pixels.

50%
100px

For this example we have the first line taking up only 10% of 135 pixels, and our second line taking up the entire 135 pixel area.

10%
135px

In this example we can see the differece between 50% and 50vw. 50% is still only 50% of the parent while 50vw is 50% of the entire viewing area.

50%
50vw
This is 10 vh or 10% of the viewing height.

px: fixed width
%: percentage of the parent
vw: one vw is 1% of the entire screen. 100 vw is the whole screen / viewing area
vh: one vh is 1% of the viewing height
